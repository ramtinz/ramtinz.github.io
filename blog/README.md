# Blog Posts

This directory contains blog posts written in Markdown format.

## Structure

- Each blog post should be saved as a `.md` file
- Use descriptive filenames with dates, e.g., `2025-01-15-my-blog-post.md`
- Include front matter at the top of each file for metadata

## Front Matter Example

```yaml
---
title: "Your Blog Post Title"
date: "2025-01-15"
author: "Ramtin Zargari Marandi"
tags: ["AI", "Machine Learning", "Healthcare"]
excerpt: "Brief description of your blog post"
---
```
